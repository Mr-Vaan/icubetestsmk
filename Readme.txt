=== TASK ===
Membuat sistem manipulasi data sederhana menggunakan PHP berdasarkan skeleton yang sudah disediakan

=== PERATURAN ===
1. Import dan gunakan database yang sudah disediakan 
2. Tidak diizinkan untuk mengubah Model.php
3. Harus implementasi Object yang sudah disediakan di Model.php untuk melakukan query ke database
4. Styling harus menggunakan external css yg disimpan di css/style.css
5. JS code harus diimplementasi di js/script.js

=== FITUR ===
1. Sistem harus bisa terhubung ke database yang sudah disediakan
2. Memiliki fitur untuk display data dari database berdasarkan struktur <table> yang ada di index.php (harus urut berdasarkan nama A-Z)
3. Memiliki fitur untuk insert data baru (tidak diijinkan terjadi duplikat data)
4. Memiliki fitur untuk update data jika data yang dimasukkan sudah ada
5. Styling halaman sesuai design yang sudah disediakan (perhatikan design guide)
6. Tombol edit bisa berfungsi untuk memasukkan data yang akan diedit otomatis ke dalam form (menggunakan js)
